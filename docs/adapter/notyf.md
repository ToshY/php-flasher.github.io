# Installation

### For Vanilla PHP 

<pre class="snippet"><code>composer require php-flasher/flasher-notyf</code></pre>

### For Laravel

<pre class="snippet"><code>composer require php-flasher/flasher-notyf-laravel</code></pre>

### For Symfony

<pre class="snippet"><code>composer require php-flasher/flasher-notyf-laravel</code></pre>
