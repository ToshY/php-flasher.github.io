# Installation

### For Vanilla PHP 

<pre class="snippet"><code>composer require php-flasher/flasher-toastr</code></pre>

### For Laravel

<pre class="snippet"><code>composer require php-flasher/flasher-toastr-laravel</code></pre>

### For Symfony

<pre class="snippet"><code>composer require php-flasher/flasher-toastr-laravel</code></pre>
