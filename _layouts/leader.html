PHPFlasher.

A powerful and flexible flash notifications system for PHP

{% include leader-links.html %}

Why use PHP Flasher ?

This library is designed, so you can take full control when creating you notifications :
  • Display multiple notifications
  • Sort and filter notifications
  • Render notification from JSON response with Ajax or Websockets
  • Limit the number of displayed notifications
  • Show notifications from different adapters at the same time
  • Framework angostic with integration for : Symfony and Laravel
  • ...and more

Easy To use to fit your taste

This library was developed with the idea that you should be able to add flash notification to your application with ease and with few lines of code.

inject the Flasher service and show a default notification library, or choose a specific one by calling the create(string $library) method.

Build you notification and add options to it using the default notification builder or a specific library builder, several builders exists to helps you create your notification.

you can also add custom informations to your notification with stamps:
The library come with some stamps out of the box :

  • CreatedAtStamp
  • PriorityStamp
  • DelayStamp
  • HopsStamp

Render you notifications

PHP Flasher make it very easy to render you notifications just by adding a simple line of code.

You dont have to worry about importing you javascript or stylesheets, Flasher handle this for you, and it only add the needed files when notifications are ready to be rendered to the browser

You can customize how you want to display you notifications: Limit the number of displayed notifications, order them and also filter them how you want with stamps, of course you can add your on logic here.

Is it all :( ?

Naaah!!! of course their is a lot more of goodies, Flasher is very powerful and flexible it allow you to do more advanced things :

  • You can add you own factories, stamps, eventSubscribers, filters .....etc.
  • Easy migration from similar libraries.
  • You can activate model observers to show notifications automatically when Eloquent events occurred.
  • Show notifications from ajax requests from a json response.
  • and lot more, just checkout the documentation
